# iterm2

## window operation

* `cmd + d` - split window vertical
* `cmd + shift + d` - split window horizontal
* `cmd + option + direction` or `cmd + {` or `cmd + }` - switch window

## tab operation
* `cmd + t` - create new tab
* `cmd + direction` or `cmd + tab-number` - switch tab

## copy and paste

* double click - copy
* `cmd + v` - paste
